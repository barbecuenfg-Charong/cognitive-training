# 注意力集中训练 (Attention Concentration Training)

本项目复刻了中国科学院心理研究所大脑机能灵敏度测试工具（舒尔特表变体）。

## 项目说明

这是一个基于Web的注意力训练游戏。玩家需要在杂乱的几何图形背景中，按顺序找出数字 1 到 100。

## 如何运行

直接使用浏览器打开 `index.html` 文件即可开始游戏。

## 游戏规则

1. 点击“开始测试”按钮。
2. 计时器开始运行。
3. 在游戏区域内，从 **1** 开始，按顺序点击数字直到 **100**。
4. 点击正确数字变灰，点击错误数字变红提示。
5. 完成后系统会根据耗时给出评级。

## 评级标准

| 等级 | 用时范围 |
| --- | --- |
| 优 (Excellent) | 15分钟以内 |
| 较好 (Good) | 15 - 20分钟 |
| 一般 (Average) | 20 - 25分钟 |
| 及格 (Pass) | 25 - 30分钟 |
| 差 (Poor) | 30 - 35分钟 |

## 技术栈

*   HTML5
*   CSS3 (Flexbox, Transformations)
*   JavaScript (ES6+)
