# 认知能力训练平台 (Cognitive Training Platform)

这是一个综合性的认知能力训练平台，旨在通过多种科学设计的游戏化任务，全面提升用户的各项认知功能。

## 1. 核心架构：十大认知系统 (The 10 Cognitive Systems)

本项目基于现代认知心理学和神经科学理论，构建了覆盖大脑核心功能的十大训练系统：

1.  **注意力系统 (Attention)**
    *   核心能力：持续性、选择性、分配性注意力
    *   代表任务：舒尔特方格、Stroop范式
2.  **记忆力系统 (Memory)**
    *   核心能力：工作记忆、短时记忆、空间记忆
    *   代表任务：N-back、科西方块 (Corsi)
3.  **执行功能 (Executive Function)**
    *   核心能力：抑制控制、认知灵活性、规划能力
    *   代表任务：Flanker任务、威斯康星卡片分类
4.  **感知觉系统 (Perception)**
    *   核心能力：视觉搜索、听觉辨别、时空感知
    *   代表任务：视觉搜索任务、音高辨别
5.  **逻辑思维 (Logical Reasoning)**
    *   核心能力：归纳推理、演绎推理、模式识别
    *   代表任务：瑞文推理测验变体
6.  **语言能力 (Language)**
    *   核心能力：词汇提取、语言理解、言语流畅性
    *   代表任务：词语接龙、语义判断
7.  **数学能力 (Mathematical)**
    *   核心能力：心算速度、数字敏感度、数量估算
    *   代表任务：快速心算、数字广度
8.  **情绪认知 (Emotional Cognition)**
    *   核心能力：情绪识别、情绪调节、同理心
    *   代表任务：面孔情绪识别
9.  **反应速度 (Reaction Speed)**
    *   核心能力：简单反应时、选择反应时
    *   代表任务：简单反应时测试
10. **空间认知 (Spatial Cognition)**
    *   核心能力：心理旋转、空间导航、空间可视化
    *   代表任务：心理旋转任务

## 2. 当前已实现功能 (Implemented Features)

目前平台已实现了以下核心训练模块：

### 2.1 舒尔特方格 (Schulte Grid) - *注意力/视觉搜索*
*   **功能**：经典的注意力训练工具，要求用户按顺序点击网格中的数字。
*   **设置**：支持 3x3 至 9x9 不同难度的网格。
*   **状态**：稳定版 (v1.0)，已移除不稳定的停止按钮功能以保证体验。

### 2.2 中科院注意力训练 (Focus Training) - *注意力/持久性*
*   **功能**：复刻中科院心理所测试，在大范围杂乱背景中寻找数字 1-100。
*   **特性**：支持缩放和平移（SVG实现）。
*   **状态**：稳定版 (v1.0)，已移除不稳定的停止按钮功能。

### 2.3 N-back 任务 - *工作记忆*
*   **功能**：经典的流体智力与工作记忆训练。
*   **设置**：支持 1-back, 2-back, 3-back 等多级难度。
*   **状态**：稳定版 (v1.0)，修复了游戏结束无响应的问题。

### 2.4 Stroop 效应测试 - *抑制控制*
*   **功能**：颜色/文字冲突测试，训练大脑的抑制控制能力。
*   **特性**：包含“停止测试”功能，支持中途退出。

### 2.5 Flanker 任务 - *选择性注意力*
*   **功能**：侧抑制任务，辨别中心箭头的方向而忽略干扰项。
*   **特性**：包含“停止测试”功能。

### 2.6 科西方块 (Corsi Block-Tapping) - *空间记忆*
*   **功能**：测试视觉-空间短时记忆广度。
*   **新特性**：
    *   **自定义方块数量**：支持设置 2-20 个方块，动态调整布局难度。
    *   **双向模式**：支持正向回忆和逆向回忆。
    *   **停止功能**：支持中途停止测试。

### 2.7 简单反应时 (Simple Reaction Time) - *反应速度*
*   **功能**：基础的视觉/听觉反应速度测试。
*   **特性**：包含麦克风/音频输入测试模块。

## 3. 技术栈 (Tech Stack)
*   **前端**：原生 HTML5 / CSS3 / JavaScript (ES6+)，无繁重框架依赖，轻量高效。
*   **图形库**：D3.js (v7) 用于复杂的数据可视化和交互（如注意力训练）。
*   **数据存储**：LocalStorage 用于保存用户偏好设置。
*   **部署**：纯静态页面，可直接部署于任何 Web 服务器或 GitHub Pages。
